# Learning Roadmap

## Purpose

This folder documents the structured learning plan, progress tracking, skill assessment, and certification goals. Focus is on deliberate skill development aligned with Junior Microsoft 365 Engineer role requirements.

---

## Goals

- Define clear learning milestones and measurable outcomes
- Track progress through operational domains
- Identify skill gaps and prioritize learning efforts
- Document certification path and preparation strategy

## Learning Phases

### Phase 1: Foundations 
- Tenant initialization and baseline configuration
- Basic identity and access management
- Fundamental security controls (MFA, Conditional Access basics)
- PowerShell connection and basic cmdlets

### Phase 2: Security & Compliance 
- Advanced Conditional Access policies
- Microsoft Defender for Office 365
- Data Loss Prevention (DLP)
- Compliance and information protection

### Phase 3: Collaboration & Productivity 
- Teams and SharePoint architecture
- External sharing and governance
- OneDrive for Business management
- Exchange Online advanced configuration

### Phase 4: Endpoint & Automation 
- Intune device management
- Compliance policies and security baselines
- PowerShell automation development
- Reporting and operational scripts

### Phase 5: Operations & Incidents
- Incident simulation and response
- Troubleshooting workflows
- Documentation and decision logs
- Portfolio preparation for job interviews

## Success Metrics

- **Operational Competence**: Ability to configure, manage, and troubleshoot M365 independently
- **Documentation Quality**: Decision logs and architecture docs ready for interview review
- **Automation Proficiency**: 10+ PowerShell scripts demonstrating practical automation
- **Incident Response**: 5+ documented incidents showing structured troubleshooting
- **Portfolio Readiness**: GitHub repository demonstrating production-level thinking

## Certification Goals

### Near-Term
- **MS-900**: Microsoft 365 Fundamentals (validates foundational knowledge)

### Medium-Term
- **MS-102**: Microsoft 365 Administrator Associate (validates operational proficiency)

### Long-Term
- **MS-500**: Microsoft 365 Security Administrator Associate (security specialization)
- **MS-700**: Managing Microsoft Teams (collaboration specialization)

## Operational Context

This is **not** passive learning. Every module, project, and certification goal is tied to demonstrable operational competence. This roadmap exists to ensure structured progress from "can follow tutorials" to "can operate production environment" — the standard for a Junior Microsoft 365 Engineer.
