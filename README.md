# Microsoft 365 Learning & Operations Lab

**Hands-on learning. Operational thinking. Enterprise mindset.**

</div>

---

## Purpose

This repository documents my **hands-on journey into Microsoft 365 operations** with a clear goal:

> **Become production-ready as a Junior Microsoft 365 Engineer.**

No theory-only notes.  
No certificate dumping.  
This repo focuses on **realistic tenant scenarios, security decisions, automation and incident handling**.

---

## What this repository represents

-  **Tenant-first thinking** instead of feature-clicking  
-  **Identity & security as core principles**  
-  **Automation where it makes sense**  
-  **Troubleshooting over tutorials**  
-  **Clear documentation and decision rationale**

This is not a demo project.  
This is a **simulated operational environment**.

---

##  Repository Structure

```text
M365-learning/
│
├── tenant_setup/                   # Initial tenant configuration & admin model
├── identity/                       # Entra ID, users, groups, access control
├── security_basics/                # MFA, Conditional Access, Secure Score
├── exchange_online/                # Mail flow, shared mailboxes, policies
├── teams_and_sharepoint/           # Collaboration & governance
├── endpoint_management/            # Intune, devices, compliance
├── user_experience/                # End-user facing services & behavior
├── powershell/                     # Admin automation & reporting
├── incidents_and_troubleshooting/  # Simulated real-world incidents
├── documentation_and_diagrams/     # Architecture, flows, decision logs
└── learning_roadmap/               # Structured learning plan, milestones and skill progression
```
